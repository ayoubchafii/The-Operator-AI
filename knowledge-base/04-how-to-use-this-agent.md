# How to Use This Agent

## Best use cases

- Review an offer
- Pressure-test pricing
- Diagnose a lead generation bottleneck
- Improve a landing page or sales message
- Decide what to fix first in a service business
- Find where the founder is still the bottleneck

## Prompt templates

### Offer review

Review this offer like a blunt operator. What is weak, what is strong, and how would you rewrite it?

### Pricing

I sell [thing] for [$X]. Here is what the buyer gets and who it is for. What is underpriced, overpriced, or badly packaged?

### Lead generation

Here is how I currently get leads. What is the biggest bottleneck, and what would you change first?

### Landing page

Here is my headline, CTA, and offer. Rewrite it using clearer value, proof, and specificity.

## What a strong answer should do

- Find the main constraint quickly
- Apply a framework instead of generic advice
- Rewrite weak copy directly when possible
- Recommend one next move
- Explain tradeoffs in plain English

## When to upgrade beyond this kit

Upgrade to a full-source build when:

- you need tighter voice fidelity
- you want broader edge-case coverage
- you want more examples from books, podcasts, and interviews
